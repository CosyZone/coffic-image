# 图像超分

import { CodeExample } from '@coffic/cosy-ui';

图像超分功能可以显著提高图片的分辨率和清晰度，让低分辨率的图片变得更加清晰和详细。AI会智能分析图片内容，生成高质量的放大版本，保持原有的细节和风格。

## 功能描述

图像超分利用AI的深度学习能力，分析低分辨率图片的像素信息和纹理特征，然后生成高分辨率的版本。与传统的简单放大不同，AI超分能够智能地"想象"和重建缺失的细节，让图片看起来更加清晰和自然。

## 使用方法

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/super_resolution_1.jpeg",
  prompt: "图像超分。",
  function: "super_resolution",
  upscaleFactor: 2
})
```text

## 参数说明

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| `imageUrl` | string | ✅ | 需要超分的图片URL |
| `prompt` | string | ✅ | 超分指令描述，说明超分的要求 |
| `function` | string | ❌ | 固定为"super_resolution" |
| `upscaleFactor` | number | ❌ | 图像超分的放大倍数，默认2 |
| `n` | number | ❌ | 生成图片数量，默认1 |
| `model` | string | ❌ | 使用的模型，默认wanx2.1-imageedit |
| `dashScopeApiKey` | string | ❌ | DashScope API密钥 |

## 提示词技巧

### 超分要求描述
- **清晰度要求**：说明对清晰度的要求，如"保持原有的清晰度"、"增强细节表现"等
- **细节保持**：说明要保持的细节，如"保持线条清晰"、"保持纹理细节"等
- **风格保持**：说明要保持的风格，如"保持艺术感"、"保持写实风格"等

### 放大倍数选择
- **2倍超分**：适合一般需求，效果明显且处理快速
- **3倍超分**：适合需要更高清晰度的场景
- **4倍超分**：适合需要大幅提升的场景，但处理时间较长
