# 模型查询

`imageedit_models` 工具用于查询图像编辑功能支持的模型列表，帮助用户了解可用的模型和它们的能力。

## 使用方法

```typescript
// 查询所有支持的模型
imageedit_models()
```text

## 返回结果

工具会返回所有支持的图像编辑模型，包括：

- **模型名称** - 模型的唯一标识符
- **模型描述** - 模型的详细说明
- **支持能力** - 模型支持的所有编辑功能
- **推荐状态** - 是否为推荐使用的模型
