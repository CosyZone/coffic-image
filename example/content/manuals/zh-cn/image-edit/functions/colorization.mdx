# 图像上色

import { CodeExample } from '@coffic/cosy-ui';

图像上色功能专门用于为黑白图片添加颜色，让历史照片、素描作品或黑白图像焕发新的生机。AI会根据提示词描述和图片内容，智能地为不同区域添加合适的颜色。

## 功能描述

图像上色功能利用AI的深度理解能力，分析黑白图片中的内容、结构和纹理，然后根据用户的提示词描述，为不同区域智能添加颜色。这种功能特别适合恢复历史照片、为素描作品上色或重新诠释黑白艺术。

## 使用方法

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/colorization_1.jpeg",
  prompt: "蓝色背景，黄色的叶子。",
  function: "colorization"
})
```text

## 参数说明

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| `imageUrl` | string | ✅ | 需要上色的黑白图片URL |
| `prompt` | string | ✅ | 上色描述，说明要添加的颜色和风格 |
| `function` | string | ❌ | 固定为"colorization" |
| `n` | number | ❌ | 生成图片数量，默认1 |
| `model` | string | ❌ | 使用的模型，默认wanx2.1-imageedit |
| `dashScopeApiKey` | string | ❌ | DashScope API密钥 |

## 提示词技巧

### 颜色描述
- **具体颜色**：明确指出要使用的颜色，如"红色"、"蓝色"、"绿色"等
- **颜色组合**：描述颜色的搭配，如"红色和黄色"、"蓝色和绿色"等
- **颜色风格**：描述颜色的风格，如"自然的"、"鲜艳的"、"柔和的"等

### 区域描述
- **主要对象**：描述主要对象的颜色，如"人物服装为蓝色"
- **背景颜色**：描述背景的颜色，如"背景为温暖的棕色"
- **细节颜色**：描述细节部分的颜色，如"眼睛为深棕色"

### 风格描述
- **写实风格**：追求自然的颜色效果
- **艺术风格**：追求艺术化的颜色表现
- **历史风格**：保持历史照片的怀旧感
- **现代风格**：使用现代的色彩搭配

### 提示词结构
```typescript
// 标准上色提示词结构
imageedit({
  imageUrl: "https://example.com/image.jpg",
  prompt: "为[主要对象]添加[颜色描述]，[背景颜色描述]，[风格描述]"
})

// 示例
imageedit({

  prompt: "为人物添加自然的颜色，服装为深蓝色，背景为温暖的棕色，保持写实风格"
})
```text
