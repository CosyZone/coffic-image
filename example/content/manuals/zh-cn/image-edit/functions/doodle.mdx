# 线稿生图

import { CodeExample } from '@coffic/cosy-ui';

线稿生图功能可以将简单的线稿、涂鸦或草图转换为完整、详细的图片。AI会智能理解线稿的内容和意图，然后生成与之匹配的完整图像，让创意想法快速变成现实。

## 功能描述

线稿生图利用AI的创意生成能力，分析线稿中的线条、形状和结构，理解用户的创作意图，然后生成与之匹配的完整图片。这种功能特别适合将创意草图快速转化为成品图像，为设计师、艺术家和创意工作者提供强大的创作工具。

## 使用方法

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/doodle_1.png",
  prompt: "北欧极简风格的客厅。",
  function: "doodle"
})
```text

## 参数说明

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| `imageUrl` | string | ✅ | 线稿图片URL |
| `prompt` | string | ✅ | 生成内容描述，说明要生成的完整图片内容 |
| `function` | string | ❌ | 固定为"doodle" |
| `n` | number | ❌ | 生成图片数量，默认1 |
| `model` | string | ❌ | 使用的模型，默认wanx2.1-imageedit |
| `dashScopeApiKey` | string | ❌ | DashScope API密钥 |

## 提示词技巧

### 风格描述
- **设计风格**：明确说明设计风格，如"北欧极简"、"现代简约"、"古典优雅"等
- **色彩搭配**：描述主要的色彩搭配，如"白色墙面"、"灰色调"、"自然色彩"等
- **材质选择**：说明使用的材质，如"木质家具"、"大理石台面"、"玻璃幕墙"等

### 细节描述
- **空间布局**：描述空间的布局和结构，如"开放式"、"大窗户"、"自然光线"等
- **装饰元素**：说明装饰和细节，如"绿色植物"、"艺术品"、"温馨灯光"等
- **氛围营造**：描述要营造的氛围，如"温馨"、"现代"、"高端"等

### 线稿质量要求
```typescript
// 线稿质量建议
1. 线条清晰：线稿的线条要清晰可辨
2. 结构明确：主要结构要明确表达
3. 比例合理：各部分的比例要合理
4. 创意明确：创意想法要明确表达
```text
