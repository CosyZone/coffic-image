# 去文字水印

import { CodeExample } from '@coffic/cosy-ui';

去文字水印功能专门用于去除图片中的水印、文字、标记等不需要的内容。AI会自动识别文字和水印区域，然后智能填充合适的内容，保持图片的完整性和美观性。

## 功能描述

去文字水印功能利用AI的智能识别能力，自动检测图片中的文字和水印区域，然后根据周围环境生成合适的填充内容，实现无缝去除效果。这种功能特别适合清理商业图片、去除版权标记或恢复图片的原始状态。

## 使用方法

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/remove_watermark_1.png",
  prompt: "去除图像中的文字",
  function: "remove_watermark"
})
```text

## 参数说明

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| `imageUrl` | string | ✅ | 需要处理的图片URL |
| `prompt` | string | ✅ | 去除指令描述，说明要去除的内容类型 |
| `function` | string | ❌ | 固定为"remove_watermark" |
| `n` | number | ❌ | 生成图片数量，默认1 |
| `model` | string | ❌ | 使用的模型，默认wanx2.1-imageedit |
| `dashScopeApiKey` | string | ❌ | DashScope API密钥 |

## 提示词技巧

### 去除内容描述
- **具体描述**：明确指出要去除的内容类型，如"水印"、"文字"、"标记"等
- **位置描述**：可以描述内容的位置，如"右下角"、"中央"、"左上角"等
- **效果描述**：描述期望的效果，如"无缝去除"、"自然过渡"等

### 保持元素描述
- **背景保持**：说明要保持的背景元素，如"保持背景的自然"、"保持画面的完整性"
- **风格保持**：说明要保持的风格，如"保持原有的艺术风格"、"保持画面的纯净"

### 提示词结构
```typescript
// 标准去除提示词结构
imageedit({
  imageUrl: "https://example.com/image.jpg",
  prompt: "去除[具体内容]，[位置描述]，[效果描述]，保持[要保持的元素]"
})

// 示例
